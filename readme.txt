This is a 48 hour project to practice creating simple AI of rock, paper and scissors with fuzzy logic influencing each teams speed.
The game can be played with no issues on it's own.
It uses three imported packages including TextMeshPro which is described in the Features.pdf file.
The options can be changed with the dropdowns and information can be seen on the right side of the game view.
The "RESET GAME" button and changing the "UNIT FORMATION CHOICE" dropdown option will reset the game.
